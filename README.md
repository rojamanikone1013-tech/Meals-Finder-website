# 🍽️ Meal Finder Website

A responsive **Meal Finder Web Application** built using **HTML, CSS, JavaScript, Bootstrap, Tailwind CSS, and TheMealDB API**. Users can search for meals, browse meal categories, and view complete recipe details including ingredients, measurements, and cooking instructions.

---

## 🚀 Features

* 🔍 Search meals by name
* 📂 Browse meals by category
* 🍽️ View all meals in a selected category
* 📖 Detailed meal information
* 🥕 Ingredients list
* 📏 Measurements for each ingredient
* 📝 Step-by-step cooking instructions
* 🏷️ Category, Area, Tags, and Recipe Source
* 📱 Fully Responsive Design
* 🎨 Modern and Clean UI
* 📋 Sidebar Category Navigation
